# Historical Evidence

This directory contains historical primary-source material relevant to the development of Mission Framework.

## Evidence status

The PDF files are the authoritative historical sources. Markdown files are derived, AI-searchable transcriptions created from the corresponding Word originals. In case of any discrepancy, the PDF controls.

The material is evidence, not current governance. It cannot override canonical documents under `docs/`.

## MIAR use

MIAR reviewers MUST first complete and freeze the public-repository review before reading this historical evidence. After reading it, they MUST record every conclusion that changed, every conclusion that became more or less confident, and every new risk or contradiction identified.

Historical confidentiality markings remain visible because they are part of the original documents. The author has explicitly authorised publication in this repository.
