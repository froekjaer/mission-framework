# Mission Framework

> Build once. Reuse forever.

## Purpose
Mission Framework is an open framework for planning, executing, documenting and preserving complex missions. It combines structured knowledge, decision support, documentation and AI-assisted workflows.

## Principles
- Facts before assumptions.
- Humans remain responsible for decisions.
- AI assists, documents and automates.
- Every important decision is traceable.
- Documentation is treated as a first-class deliverable.
- Design for reuse.

## Vision
Create a framework that can support projects such as Mission Eclipse, maritime expeditions, timelapse systems, research projects and future missions using a common architecture.

## Motto
Build once. Reuse forever.
