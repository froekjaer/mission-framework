# Known Open Questions

Mission Framework v0.2 does not resolve the following questions.

## Conceptual

- Which concepts are genuinely useful additions, and which duplicate established disciplines?
- Can Collaborative Intelligence be defined with sufficient precision to support reproducible practice?
- How should the framework distinguish understanding, knowledge, justified reliance and decision quality?
- What would falsify the central propositions of Mission Framework?

## Methodological

- Which minimum artefacts are necessary for a mission to remain reviewable without becoming bureaucratic?
- How should uncertainty, contradiction and negative results be represented consistently?
- How can outcomes be attributed when several people, systems and environmental factors interact?
- Which measures can show that understanding or decisions improved?

## Governance

- Which decisions require human accountability and which may be delegated under bounded authority?
- How should minority views and unresolved contradictions be preserved?
- How can governance resist authority bias, groupthink and capture by founders or institutions?
- When should a concept be revised, deprecated or removed from Mission Core?

## Practical application

- How well does the framework transfer between public, commercial, scientific and personal missions?
- What is the smallest useful implementation?
- Where does the framework impose more cost than value?
- Which failure modes emerge only during long-running missions?

Reviewers are encouraged to add questions rather than only answer those already listed.
