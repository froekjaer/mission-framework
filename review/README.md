# Mission Framework v0.2 Review Candidate

Mission Framework v0.2 is released as an open Review Candidate for independent assessment.

The purpose of this review is not endorsement. It is to determine whether the framework contributes useful concepts, methods or questions, and to identify where it is unclear, unsupported, incomplete, overlapping or wrong.

## Review tracks

Reviewers may contribute through one or more tracks:

- Academic review
- Practitioner review
- AI research and governance review
- Structured AI-assisted review

## Participation

Participation should not be interpreted as endorsement of Mission Framework, its contributors or its conclusions.

Reviewers are invited to challenge definitions, assumptions, methods, evidence, scope, governance and practical applicability.

## Suggested starting points

1. Read the repository README and canonical glossary.
2. Review the current limitations and open questions.
3. Select the review guide most relevant to your background.
4. Submit comments using the review template or a GitHub issue.

The review process is part of the framework itself: criticism, contradiction and replacement are expected contributions to learning.
