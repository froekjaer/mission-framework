# Publication Profiles

This directory contains declarative publication profiles and supporting configuration for the Mission Framework Publication Pipeline.

The normative model is defined in [`docs/publication/PUBLICATION-PIPELINE.md`](../docs/publication/PUBLICATION-PIPELINE.md).

Initial profiles:

- `book.yml`
- `article.yml`
- `executive-brief.yml`
- `review-package.yml`
- `presentation.yml`
- `github-pages.yml`

The profiles are intentionally minimal while the publication generator is under review. They define purpose, audience, source intent, metadata requirements and expected outputs. They do not yet claim that all listed formats can be rendered.
